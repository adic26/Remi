Git Integration Tips for Visual Studio 2012

1) Download Visual Studio Tools for Git here: http://visualstudiogallery.msdn.microsoft.com/abafc7d6-dcaa-40f4-8a5e-d6724bdb980c/file/93137/7/Microsoft.TeamFoundation.Git.Provider.msi

2) In the Visual Studio menu, click TOOLS -> Options. Expand the Source Control node and select Plug-in Selection. Change the Current source control plug-in to Microsoft Git Provider

3) In the Visual Studio menu, click VIEW -> Team Explorer. Click the drop-down menu and select Settings -> Git Settings. Enter your information and click Update

 

Once you restart Visual Studio, your Solution Explorer should display icons (blue lock symbols or red chack marks) to show file status.

The Team Explorer window can be used to view changes, manage branches, make commits, etc.

Please note that the Visual Studio plug-in does not support transfers using the SSH protocol, so cannot push/pull to/from GitLab remote repositories.

 

The following steps are optional:

4) In the Visual Studio menu, click TOOLS -> External Tools. Click the Add button to create three custom tools with the following information:

Tool #1:

Title: Git Bash

Command: C:\Program Files (x86)\Git\bin\sh.exe

Arguments: --login -i

Initial directory: $(SolutionDir)

Additional: Close on exit

Tool #2:

Title: Git Status

Command: C:\Program Files (x86)\Git\bin\git.exe

Arguments: status

Initial directory: $(SolutionDir)

Tool #3 (note that git push will only work if your local branch is tracking a remote branch)

Title: Git Push

Command: C:\Program Files (x86)\Git\bin\git.exe

Arguments: push

Initial directory: $(SolutionDir)

Additional: Use Output window, Close on exit

5) Use the Move Up button to move the three newly created tools to the top of the list

6) In the Visual Studio menu, click VIEW -> Toolbars -> Customize... Click New and name the new toolbar Git

7) Click the Commands tab, select the Toolbar radio button and change the drop-down menu to Git

8) Click Add Command... In the Categories list, select Tools. In the Commands list, select External Command 1

9) Repeat step (8) to add External Command 2 and External Command 3

10) Use the Move Up and Move Down buttons to rearrange the External Commands in ascending order. You should now see a new toolbar near the top of your Visual Studio IDE, with buttons for Git Bash, Git Status and Git Push
